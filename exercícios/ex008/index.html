Principais formatações

1. Negrito/Destaque

Nessa frase, temos um Termo em negrito usando a tag <b> e </b> ( NÃO SEMÂNTICO )

Nessa frase, tenho um Termo em destaque usando a tag <strong> </strong> ( SEMÂNTICO )

2. Itálico/Ênfase

Nessa frase, temos um termo em itálico usando a tag <i> e </i> (NÃO SEMÂNTICO)

Nessa frase, temos um termo em ênfase usando a tag <em> e </em> ( SEMÂNTICO )

3. Texto marcado

Podemos criar tambem um texto marcado usando a tag <mark> e </mark>

Temos outro texto marcado no final

4. Texto grande e pequeno

Estamos fazendo um texto grande e um texto queno. o grande com <big> e </big>, ja o pequeno <small> e </small> ( BIG - NÃO SEMÂNTICO. SMALL - SEMÂNTICO )

5. Texto deletado

Podemos marcar um texto deletado com a tag <del> e </del>

6. Texto inserido

Podemos marcar um texto inserido com a tag <ins> e </ins>

Tambem temos o texto com sublinhado com a tag <u> e </u> ( NÃO SEMÂNTICO )

7. Texto sobrescrito

Para inserir coisas do tipo x2+4x-2=0 utiliza-se a tag <sup> e </sup>

8. Texto subscrito

Para inserir coisas do tipo H2O utiliza-se a tag <sub> e </sub>
